# CSE368 Project - Group 10
- Project files contained in CSE368_Project folder, training data in CSE368_Project folder/Training Data
- The project will not run without the .env file (included with the demo submission) in the CSE368_Project folder.
- Must have docker desktop installed to run the server and the database.
- Use command 'docker compose up --build --force-recreate' while in the CSE368_Project folder to run the app and navigate to given url

Tech used:
- Flask for basic app framework
- MongoDB for database
- Gemini AI and Vertex AI APIs for content generation and supervised tuning
- Mistral AI for OCR
